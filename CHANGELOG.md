# Changelog

All notable changes to this project are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and the project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-03-31

### Added

- Make.com scenario with Shopify trigger and automatic distribution
- Telegram Bot API integration (2 channels, photo/carousel/video support)
- Facebook Pages Graph API integration (single photo, carousel, video)
- Instagram Business Graph API integration (single photo, carousel, Reels)
- Standalone Python script with Streamlit web interface
- WhatsApp Cloud API module (structure ready, pending Groups API stabilization)
- Complete setup guide for each service
- Configuration file with `.env` support
- Professional README with architecture diagrams and documentation

### In Development

- Full WhatsApp Groups integration
- Advanced multi-photo carousel support on Make.com
- Publication history dashboard
- Error notifications via email
- Scheduled posting feature
